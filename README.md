# Invoiça Mobile

Application mobile React Native pour la gestion de devis et factures artisans.

## Stack

- Expo SDK 55
- React Native 0.83
- Expo Router (file-based routing)
- TypeScript
- Axios

## Démarrage

```bash
npm install
npx expo start
```

## Connexion à l'API

L'application se connecte à l'API Laravel Invoiça sur `http://localhost:8000/api/v1`.
